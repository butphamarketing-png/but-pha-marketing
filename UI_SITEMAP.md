## 5. Primary User Flows

### 5.1 Create content
Dashboard -> New Article Wizard -> Outline Review -> Draft Editor -> SEO Check -> Publish Queue

### 5.2 Refresh content
Dashboard alert -> Article Detail -> Refresh Draft -> Review diff -> Approve -> Publish Queue

### 5.3 Investigate rank drop
Dashboard alert -> Keyword detail -> Rank history -> SERP analysis -> Refresh draft

### 5.4 Build topical cluster
Clusters -> Generate cluster -> Review pillar + cluster topics -> Add articles
